# Hospital Management System

A **Spring Boot** project for managing hospital operations including **patients, doctors, appointments, and billing**. This project exposes RESTful APIs which can be consumed by frontend apps or mobile applications such as React Native.

## Features

- CRUD operations for:
  - **Patients**  
  - **Doctors**  
  - **Appointments**  
  - **Bills**  
- Logging with **Log4j2** for all service operations  
- RESTful APIs for mobile or web consumption  
- Database persistence with **JPA & MySQL**  

---

## Technologies

- **Backend**: Java 17, Spring Boot  
- **Database**: MySQL  
- **Persistence**: JPA / Hibernate  
- **Logging**: Log4j2  
- **Build Tool**: Maven  
- **Version Control**: Git / GitHub
